Tipos de overflow

overflow se refere ao comportamento de conteúdo que ultrapassa os limites de um contêiner. Em vez de permitir que esse conteúdo "transborde" de forma desorganizada, o CSS oferece várias maneiras de lidar com ele. O overflow controla o que acontece quando o conteúdo de um elemento excede suas dimensões especificadas (largura e altura).

overflow: visible

é o padrão, mesmo que o conteudo fique fora do elemento ele ira mostrar

overflow:hidden

Comportamento: O conteúdo que excede o tamanho do contêiner será ocultado (não será visível). ex:

.caixa {
width: 200px;
height: 100px;
overflow: hidden;
}

overflow:scroll

Comportamento: O conteúdo que excede o tamanho do contêiner será cortado, mas barras de rolagem (tanto horizontais quanto verticais) serão adicionadas para permitir que o usuário role e veja o conteúdo oculto, ex:

.caixa {
width: 200px;
height: 100px;
overflow: scroll;
}

overflow:auto

Comportamento: O conteúdo será cortado, e as barras de rolagem serão adicionadas automaticamente, somente quando necessário. Ou seja, se o conteúdo não ultrapassar as dimensões do contêiner, não aparecerão barras de rolagem.

.caixa {
width: 200px;
height: 100px;
overflow: auto;
}

overflow-y:hidden

display-flex

display: flex é uma propriedade do CSS que transforma um elemento em um container flexível (“flex container”).

Dentro dele, todos os elementos filhos se tornam itens flexíveis (“flex items”).

Isso facilita muito organizar elementos em linha, centralizar coisas, criar espaçamentos automáticos, e deixar o layout responsivo sem complicação.

Exemplo:
.container {
display: flex;
flex-direction: row; /* horizontal (padrão) */
/* flex-direction: column; vertical */ }
