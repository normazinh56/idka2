Fontes de texto

o body no css significa todo o corpo do site então se voce definir um tipo de fonte dentro do body todo o site
vai usa somente esse tipo de fonte de texto, para usar o body veja um exemplo abaixo

body{
background-color: #ccc;
font-family: Arial, sans-serif;
}

Tipos de Estilo de Borda (border-style)

Estilo Exemplo Descrição
none sem borda Não mostra borda (padrão).
hidden invisível Igual ao none, mas usado em tabelas.
solid ───── Linha contínua.
dashed - - - - Linha tracejada.
dotted . . . . Linha pontilhada.
double ════ Dupla linha.
groove efeito 3D Parecido com “entalhado”.
ridge efeito 3D Inverso do groove.
inset efeito 3D Afundado.
outset efeito 3D Elevado.

Propriedades Gerais de Borda

Propriedade Exemplo Descrição
border-width thin medium thick 2px Define a espessura da borda.
border-color border-color: red; Define a cor da borda.
border-style solid, dashed, etc. Define o estilo da borda.
border border: 2px solid #000; Atalho que combina espessura, estilo e cor.

Tipos de fontes no css

font-family, Define a família de fontes usada no elemento. font-family: "Roboto", Arial, sans-serif;
font-size, Define o tamanho da fonte. font-size: 16px;
font-weight, Define o peso (espessura) da fonte. font-weight: 300;
font-style, Define o estilo da fonte. font-style: italic;
font-variant, Define variações tipográficas como small-caps. font-variant: small-caps;
text-decoration, Decoração do texto. text-decoration: underline;
text-align, Alinhamento do texto. text-align: center;
text-indent, Recuo da primeira linha. text-indent: 30px;
text-shadow, Sombra no texto. text-shadow: 2px 2px 5px #000;
white-space, Controle de espaçamento e quebras. white-space: nowrap;

Tipos de espaçamentos

Aqui vão todos os tipos de espaçamentos:

margin: 10px; — Todas as margens iguais
margin: 10px 20px; — Cima/Baixo = 10px, Esquerda/Direita = 20px
margin: 10px 20px 30px; — Cima = 10px, Lados = 20px, Baixo = 30px
margin: 10px 20px 30px 40px; — Cima, Direita, Baixo, Esquerda (sentido horário)
margin-top: 15px; — Espaço superior do elemento
margin-right: 20px; — Espaço à direita
margin-bottom: 25px; — Margem inferior
margin-left: 30px; — Margem à esquerda
margin: auto; — Centraliza o elemento horizontalmente
padding Ele “empurra” o conteúdo para dentro do elemento, aumentando o espaço entre o conteúdo
